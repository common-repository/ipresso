iPresso is a complex Marketing Automation 360 platform. This plugin is dedicated to the clients of iPresso.

== Description ==

iPresso is a complex Marketing Automation 360 platform, integrating the most important tools for successful marketing.

iPresso Plugin adds marketing consents into comment and register forms.
It collects correct permissions and sends them as new marketing contacts to iPresso.

User may choose which consents will be shown on the site and see which activities have their consents checked.

== Installation ==

1. Upload `ipresso` folder to the `/wp-content/plugins/` directory.

2. Activate the plugin through the 'Plugins' menu in WordPress.

3. Register your plugin using iPresso Panel URL address, username, password and customer key.
